# langchain-提示词模板

注意：使用提示词模板时，需要注意的是生成的提示词模板都是为Promise对象，需要使用await来等待结果。

关于字符串模板的使用：
    在定义字符串模板的时候，使用引号(ts不区分单双引号)或者反引号来包裹模板内容。
    引号：不可格式化，不可换行。
    反引号：可格式化，可换行。
    所以我们做字符串模板的时候，尽量使用反引号来包裹模板内容。
    例如：

```ts
   const template = `
    <template>
    <div>
    <h1>Hello, World!</h1>
    </div>
    </template>
 `;
```


## 1.字符串型提示词模板

PromptTemplate

> fromTemplate方法：从字符串模板创建提示词模板。
>
> format方法：将提示词模板中的变量替换为具体的值。

```ts
export async function stringPrompt(name: string) {
  const template =
    PromptTemplate.fromTemplate(`你是一名作家，你的名字叫做{name}`);
  const prompt = await template.format({ name });
  return prompt;
}
```

控制台打印

```
你是一名作家，你的名字叫做张三
```

## 2.对话型提示词模板

ChatPromptTemplate

> fromMessages方法：从消息列表创建提示词模板。
>
> formatMessages方法：将提示词模板中的变量替换为具体的值。

```ts
export async function chatPrompt(Obj: { name: string; input: string }) {
  const template = ChatPromptTemplate.fromMessages([
    ["system", "你是一名作家，你的名字叫做{name}"],
    ["human", "你好，你现在在哪"],
    ["ai", "你好，我在上海"],
    ["human", "{input}"],
  ]);
  const prompt = await template.formatMessages(Obj);
  return prompt;
}
```

## 3.对话型提示词模板拆分

  ChatPromptTemplate是一个数组，有一个个元素，每个元素是一个消息，消息的格式为[role, content]。

  我们可以将其拆分后在组合成一个完整的对话型提示词模板。

  该效果等同于ChatPromptTemplate，只是该方法更加的灵活，可以更方便的组合提示词模板。

```ts
export async function chatPromptSplit(Obj: { name: string; input: string }) {
  const systemMessage = SystemMessagePromptTemplate.fromTemplate(
    "你是一名作家，你的名字叫做{name}",
  );
  const humanMessage = HumanMessagePromptTemplate.fromTemplate("{input}");
  const aiMessage = AIMessagePromptTemplate.fromTemplate("你好，我在上海");
  const template = ChatPromptTemplate.fromMessages([
    systemMessage,
    humanMessage,
    aiMessage,
  ]);
  const prompt = await template.formatMessages(Obj);
  return prompt;
}
```

## 4.自定义角色对话提示词模板

ChatMessagePromptTemplate

AIMessagePromptTemplate,HumanMessagePromptTemplate,SystemMessagePromptTemplate 这三个方法为系统提供的默认角色提示词模板，为了更加灵活，我们可以使用ChatMessagePromptTemplate来实现自定义角色提示词模板。

```ts
export async function customRolePrompt(Obj: { name: string }) {
  const template = ChatMessagePromptTemplate.fromTemplate(
    "你是一名作家，你的名字叫做{name}",
    "主角",
  );
  const prompt = await template.format({ name: Obj.name });
  return prompt;
}
```

控制台打印

```

  "additional_kwargs": {},
  "response_metadata": {},
  "role": "主角"
}
```

## 6.组合提示词模板

PipelinePromptTemplate允许你将多个提示模板组合成一个流水线。这在构建复杂提示（例如包含系统消息、历史记录和用户输入的对话提示）时非常有用。

>   pipelinePrompts字段：将多个提示词模板组合成一个数组。
>
>   finalPrompt：最终的提示词模板。

```ts
export async function pipelinePrompt() {
  const systemMessage = PromptTemplate.fromTemplate("你是{role}助手");
  const historyMessage =
    PromptTemplate.fromTemplate("你是对话历史为：{history}");
  const userMessage = PromptTemplate.fromTemplate("用户提问为：{input}");
  const finalPrompt = "{system}\n{history}\n{user}\n请给出回复：";
  const pipeline = new PipelinePromptTemplate({
    pipelinePrompts: [
      { name: "system", prompt: systemMessage },
      { name: "history", prompt: historyMessage },
      { name: "user", prompt: userMessage },
    ],
    finalPrompt: PromptTemplate.fromTemplate(finalPrompt),
  });
  const prompt = await pipeline.format({
    role: "智能",
    history: "你好，我现在在哪",
    input: "帮我订机票",
  });
  console.log(prompt);

  return prompt;
}
```

控制台打印

```
你是智能助手
你是对话历史为：你好，我现在在哪
用户提问为：帮我订机票
请给出回复：
```

## 7.序列化提示词模板

使用 JSON 文件存储提示词模板，通过 PromptTemplate.deserialize 加载。

JSON 格式需包含：input_variables, template, template_format (可选)

```ts
import { readFile } from "fs/promises";
export async function serializePrompt() {
  const jsonString = await readFile("src/promt/12.json", "utf-8");
  const data = JSON.parse(jsonString) as SerializedPromptTemplate;
  const loadedPrompt = await PromptTemplate.deserialize(data);

  const formatted = await loadedPrompt.format({
    subject: "猫",
    adjective: "有趣",
  });
  console.log(formatted);
  return formatted;
}
```

json文件

```ts
{
  "input_variables": ["subject", "adjective"],
  "template": "给我讲一个关于{subject}的{adjective}故事。",
  "template_format": "f-string",
  "type": "prompt"
}
```

控制台打印

```
给我讲一个关于猫的有趣故事。
```
