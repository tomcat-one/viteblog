# langchain-示例选择器

## 1.长度动态提示词

根据输入长度动态选择示例数量，避免超过 maxLength。

输入越长，选入的示例越少；输入越短，示例越多；

 LengthBasedExampleSelector：根据输入长度动态选择示例数量，避免超过 maxLength；

  FewShotPromptTemplate：根据示例选择器和示例提示词模板，生成提示词模板；

```ts
import { PromptTemplate, FewShotPromptTemplate } from "@langchain/core/prompts";
import {
  LengthBasedExampleSelector
} from "@langchain/core/example_selectors";
export async function lengthBasedDynamicPrompt(word: string) {
  const examples = [
    { input: "happy", output: "sad" },
    { input: "tall", output: "short" },
    { input: "sunny", output: "gloomy" },
    { input: "windy", output: "calm" },
    { input: "高兴", output: "悲伤" },
    { input: "炎热", output: "寒冷" },
  ];

  const examplePrompt = new PromptTemplate({
    inputVariables: ["input", "output"],
    template: "原词：{input}\n反义：{output}",
  });

  const selector = await LengthBasedExampleSelector.fromExamples(examples, {
    examplePrompt,
    maxLength: 300,
  });

  const dynamicPrompt = new FewShotPromptTemplate({
    exampleSelector: selector,
    examplePrompt,
    prefix: "请根据示例给出每个输入词的反义词。",
    suffix: "输入: {word}\n输出:",
    inputVariables: ["word"],
  });
  console.log("dynamicPrompt", await dynamicPrompt.format({ word }));
}
```

## 2.语义相似度动态提示词

根据输入与示例的语义相似度选择最相关的 k 个示例。

使用通义千问 Embeddings（DashScope），模型使用text-embedding-v3

```ts
import { PromptTemplate, FewShotPromptTemplate } from "@langchain/core/prompts";
import {SemanticSimilarityExampleSelector} from "@langchain/core/example_selectors";
import { FakeVectorStore } from "@langchain/core/utils/testing";
import { OpenAIEmbeddings } from "@langchain/openai";
function createQwenEmbeddings() {
  return new OpenAIEmbeddings({
    configuration: {
      baseURL: "https://dashscope.aliyuncs.com/compatible-mode/v1",
    },
    apiKey: "sk-xxx",
    model: "text-embedding-v3",
  });
}
export async function semanticSimilarityDynamicPrompt() {
  const examples = [
    { input: "happy", output: "sad" },
    { input: "tall", output: "short" },
    { input: "energetic", output: "lethargic" },
    { input: "sunny", output: "gloomy" },
    { input: "windy", output: "calm" },
    { input: "高兴", output: "悲伤" },
    { input: "炎热", output: "寒冷" },
  ];

  const embeddings = createQwenEmbeddings();
  const exampleSelector = await SemanticSimilarityExampleSelector.fromExamples(
    examples,
    embeddings,
    FakeVectorStore,
    { k: 2 },
  );

  const dynamicPrompt = new FewShotPromptTemplate({
    exampleSelector,
    examplePrompt: new PromptTemplate({
      inputVariables: ["input", "output"],
      template: "输入: {input}\n输出: {output}",
    }),
    prefix: "请根据示例给出每个输入词的反义词。",
    suffix: "输入: {word}\n输出:",
    inputVariables: ["word"],
  });

  //   return dynamicPrompt.format({ word });
  console.log("dynamicPrompt", await dynamicPrompt.format({ word: "高兴" }));
}

```

```

输出: 悲伤

输入: happy
输出: sad

输入: 高兴
输出:
```
